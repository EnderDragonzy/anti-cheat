# Anti-Cheat

## Deploy

First, install npm. Then configure your apiKey in config.js:

```js
let apiKey = 'your-key-here'
```

Then initialize & run the bot with:

```bash
npm i                # first time only
npm run anticheat
```

After login, you should see it works, just send it 'ding' for test.
